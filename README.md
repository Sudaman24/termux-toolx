 ### 如何使用？
 输入特定输出的数字：
 - (1): 显示所有可用的工具并输入要安装的工具的编号。
 - (2): 显示工具类别。
 - (3): 如果你想更新 Tool-X。
 - (4): 如果您想了解我们。
 - (x)：退出工具。
 <br/>
 从终端中的任何位置键入“Tool-X”或“toolx”以打开 Tool-X。
